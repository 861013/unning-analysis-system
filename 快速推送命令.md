## 方式二：手动推送（使用Token）

### 步骤1：创建Personal Access Token

1. 访问：https://github.com/settings/tokens
2. 点击 "Generate new token" → "Generate new token (classic)"
3. 填写信息：
   - Note: `本地开发`
   - Expiration: 选择过期时间
   - Select scopes: 勾选 `repo`
4. 点击 "Generate token"
5. **立即复制token**（只显示一次）

### 步骤2：执行推送命令

```bash
cd /home/hadoop/nosql/run/demo

# 将 YOUR_TOKEN 替换为您刚才复制的token
git remote set-url origin https://861013:YOUR_TOKEN@github.com/861013/unning-analysis-system.git

# 推送
git push -u origin main
```

## 方式三：使用凭据助手（需要交互输入）

```bash
cd /home/hadoop/nosql/run/demo

# 配置凭据助手（只需执行一次）
git config --global credential.helper store

# 推送（会提示输入用户名和密码）
git push -u origin main
```

当提示时：
- **Username**: `861013`
- **Password**: 粘贴您的Personal Access Token
